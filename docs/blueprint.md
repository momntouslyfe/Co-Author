# **App Name**: Co-Author Pro

## Core Features:

- User Authentication: Google OAuth based authentication with Firestore user document creation and affiliate ID generation.
- Monetization System: Hybrid PAYG and subscription model supporting Stripe, Paddle, and UddoktaPay with coupon code management.
- Affiliate System: Automated affiliate tracking and commission calculation with a dedicated user dashboard and admin management.
- AI Topic Research Assistant: Deep topic research, pain point analysis, target audience suggestion tool for enriched book content and project blueprint context.
- AI Blueprint Generator: AI powered book structure generation with distinct, complete outlines based on market analysis, research and user input.
- AI Co-Author Workspace: Main writing area where the book is created, chapter by chapter, with content expansion enabled using an Extend with AI tool, ensuring high quality and adherence to the user's selected style.
- Built-in Blog System: Admin-managed blog system with SEO features like sitemap generation, Google Search Console integration, and analytics tracking via GA4 and Facebook Pixel/CAPI.
- Backend Multiple AI Integration: Like Open AI, Gemini, Claude etc for all the ai tasks. Admin can provide API Key and Integrate them, Also Admin can select Specific AI with Model they provide and use single or multiple AI services for All AI Functionality of the App.

## Style Guidelines:

- Primary color: Deep indigo (#10B981) to evoke a sense of knowledge, depth, and creativity.
- Background color: White for a clean and calming backdrop.
- Accent color: Electric Blue (#3B82F6) to provide contrast and highlight key elements.
- Headline font: 'Playfair', serif. Body Font: 'PT Sans', sans-serif, for a balance of elegance and readability.
- Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets.
- Consistent use of modern, flat icons across the platform to enhance usability.
- Clean and intuitive layout with a focus on readability and user experience.
- Subtle animations and transitions to provide a smooth and engaging user experience.
